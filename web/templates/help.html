Help

How to use {{ app_name }} and manage your local tracker data.

About

{{ app_name }} is a local run tracker for The Bazaar. It records runs, tracks achievements, and stores data locally on your computer.

The tracker need to be running while you play for automatic import of the runs.

Runs need to be marked as verified. Confirm that that the data is correct and mark the runs as verified.

Known Limitations

  • Only works on windows, with steam game installation.
  • Needs to play the game in 1920x1080 resolution.
  • Spawned items(from other items, or events) are not automaticcally recorded in the run.
  • A run with 10 wins and at least 25 prestige counts as a perfect run.

First setup

  1. Open the Manage page.
  2. Update the template database from the game’s cards.json.
  3. Update item images. It's normal for it to take a long time to get all images.
  4. Optionally import checklist progress from CSV.

Recording runs

Runs can be added in different ways:

  • automatic import from logs
  • manual creation
  • screenshot-based OCR

Any run can be edited later.

Achievements

Achievements unlock automatically based on your run history.

They are rebuilt from stored data, so progress stays consistent even after edits.

Board and item data

Item templates are imported from the game’s cards.json.

Item images can be downloaded and cached locally from the Manage page.

Backups

Use the Manage page to:

  • export runs as JSON
  • export a full backup
  • import a previous backup

Data location

All tracker data is stored locally.

You can check the current data directory on the Manage page.

Troubleshooting

If something looks wrong, open the Manage page and run Doctor.

That helps check local tracker files and database consistency.
